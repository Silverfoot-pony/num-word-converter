# num-word-converter
Small QT6 application to convert nums to words and vice versa.
Just something I vibecoded together while testing an Ollama model. Uses QT6.11. Designed for EndeavourOS and KDE Plasma 6.6.3. Works on my machine and if it doesnt on yours go fix it yourself.
Run pip install PyQt6 word2number num2words first so you have the packages.
